## Practica 1.2 Metodo Burbuja Avanzado
### Fecha; 21/04/2026
### Descripcion:
Cree la clase ´Metodo Burbuja Avanzado´ que implementa el método de ordenamiento burbuja con una optimización clave: el uso de la variable booleana huboIntercambios. Esta variable se inicializa en false al comienzo de cada pasada; si durante el recorrido del arreglo no se realiza ningún intercambio, significa que el arreglo ya está ordenado y el algoritmo termina anticipadamente mediante un break, evitando comparaciones innecesarias. Esto reduce el número de iteraciones en casos donde el arreglo se ordena antes de completar todas las pasadas, mejorando el rendimiento respecto al método burbuja simple.
